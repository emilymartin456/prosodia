# 安全策略

## 支持的版本

| 版本 | 是否支持 |
|------|----------|
| 0.1.x | ✅ |

项目尚在早期（0.x），安全修复只在最新的次版本上进行。

## 报告漏洞

请**不要**通过公开 issue 报告安全问题。

请使用 GitHub 的私密漏洞上报（仓库 **Security → Report a vulnerability**）。
请附上复现步骤与影响范围。我会在 5 个工作日内回应，并在修复发布后与你商定披露时间。

## 范围说明

`prosodia` 会加载并解析文本、读写 WAV 文件。需要留意的方向包括：畸形输入导致的异常/拒绝服务、
以及可选的 `openai` 适配器发起的网络请求（默认关闭，需显式启用）。核心链路离线、不执行外部代码。
